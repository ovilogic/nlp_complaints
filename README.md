# Customer Complaints Intelligence

## Project Overview

A project to analyse customer complaints across financial products and identify common themes, sentiment trends, and operational pain points. It applies natural language processing (NLP) techniques and interactive visualisations to transform unstructured customer feedback into actionable business insights.

## Business Questions

- Which products generate the highest complaint volumes?
- Which products are associated with the most negative customer sentiment?
- What themes recur most frequently in customer complaints?
- How distinct are complaint patterns between financial products?
- Can complaint text alone be used to predict product categories?

## Dataset

- Source: Kaggle https://www.kaggle.com/datasets/shashwatwork/consume-complaints-dataset-fo-nlp;
- Size: ~162k complaints;
- Key fields: product, narrative.
